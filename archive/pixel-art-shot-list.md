# Pixel Art Shot List (Production Assets)

This list targets MakeCode Arcade on PyGamer (160x120, 16-color palette). Temporary placeholders are already embedded in `main.ts`.

## Core Characters (16x16)
- Winston camel idle/walk (2-4 frames), plus vortex reaction frame.
- Nolan Bushnell NPC portrait + in-scene sprite.
- Toru Iwatani NPC portrait + in-scene sprite.
- Alexey Pajitnov NPC portrait + in-scene sprite.

## Chapter 0: Present Day / Endings
- Game jam exterior tile set (8x8 tiles): building, door, sign, sidewalk.
- Vortex animation (16x16 or 24x24, 4-6 frames).
- 2026 ending background panel.
- 2030 ending variant panel (futuristic sign, fireworks accent sprites).

## Chapter 1: Breakout Era (Atari 1975)
- Office tile set (8x8): wood paneling, desk, CRT monitor, floor.
- Paddle variants (20x3): proto, improved, final.
- Ball sprites (4x4) and hit flash effect.
- Brick set (16x6): monochrome row + colored score rows.

## Chapter 2: Maze Era (Namco 1979)
- Office tile set (8x8): clean workstation, wall accents.
- Pac-style player (8x8) with mouth-open/closed frames.
- Ghost set (8x8): red, pink, cyan, orange + frightened blue + eyes-only.
- Dot/pellet sprites (3x3 and 5x5) + pellet pulse frame.

## Chapter 3: Falling Blocks Era (Moscow 1984)
- Soviet office tile set (8x8): terminal, desk, muted wall/floor.
- Elektronika 60 panel/background frame (UI overlay).
- Pentomino icon sheet (5-cell forms) for tutorial/dialog callouts.
- Tetromino block skin variants (single-cell 7x7 draw style reference).
- Line-clear flash strip (8x1 frames or palette-swap plan).

## UI / Meta
- Title logo (up to 128x32).
- Dialogue box frame tiles (corners/edges/fill).
- Locked/crossed option icons (8x8).
- Chapter cards: 1975, 1979, 1984 (small banner art).
